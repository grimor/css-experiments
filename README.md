# CSS Experiments

A collection of interactive experiments demonstrating mathematical CSS concepts.

## Examples

### Card Stack

An interactive stacked card layout that uses CSS math to create a 3D card fanning effect. Control the angle, gap between cards, and rotation using the debug panel. Cards are positioned and rotated using CSS variables and mathematical calculations.

### Counter

A digital counter animation inspired by flip clocks and odometers. Features multiple rotating segments that spin at different speeds to display changing numbers. Uses CSS transformations and animations combined with mathematical timing calculations for a smooth counting effect.

### Repeller

An interactive effect where elements react to your mouse position. Move your cursor around the screen to see how the particles or visual elements repel away from the mouse pointer. Demonstrates real-time coordinate tracking and distance-based calculations in CSS.

### Spotlight

A dynamic spotlight that follows your mouse movement across the screen. Creates an interactive illumination effect where a spotlight reveals or highlights content as you move. Uses mouse coordinates to calculate spotlight position using CSS math functions.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
